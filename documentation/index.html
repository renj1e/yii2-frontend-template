Overview

Bravana is multipurpose website template for business, agency, e-commerce or even for blogging. Beside it's full responsive, it also has a lot of features that will save your time building website (premium Slider Revolution included, you save $14). The theme is built using Sass with very well managed Sass file structure, so you can learn and customize the theme faster. Each page in the theme is handcrafted with care in terms of design and performance, such as we always include necessary javascript file only. Although the script/code is self-explanatory, we provide you with detailed theme documentation. However, you can always reach us whenever you need help to work with the theme and it's free support. Lastly, once you buy the theme, you are eligible for future free updates forever.

Important Notes

Recommended: If you don't use Sass, it's recommended that you write your own styles to file assets/css/my-custom-styles.css. Avoid any direct change to main.css for easier theme update release in the future.
Note: Newsletter and contact form only works in web server with PHP environment. We recommend you to develop the theme in web server, i.e. XAMPP, MAMP or others.
List of important files:

Below is list of files that you might need to look and work on mostly.

  • assets/css/main.css (essential/primary style for the theme)
  • assets/css/main-shop.css (specific styles for shop templates)
  • assets/css/special-*.css (specific styles for special thematic templates, *such as restaurant, education)
  • assets/css/my-custom-styles.css (your custom styles here, can be used to override exisiting styles)
  • assets/js/bravana.js (all of the plugin initialization scripts and others DOM manipulation such as for navigation)
  • assets/js/bravana-shop.js (specific plugin initialization scripts and others DOM manipulation for shop templates)
  • sass/main.scss (Sass file for main.css)
  • sass/main-shop.scss (Sass file for main-shop.css)
  • sass/special-*.scss (Sass files for special thematic templates, *such as restaurant, education)
  • php/contact.html (PHP file to process and send email from contact form)
  • php/mailchimp.html (PHP file to process newsletter subscription)

Twitter Feed

Bravana has working Twitter feed at sidebar implemented with Twitter Post Fetcher plugin. Please see below for the guides.

You can find how the plugin is configured and initialized at assets/js/bravana.js

This configuration uses custom templating and custom date formatter for tweet output. The main concern here is "id" parameter, please follow these steps to get your id:

  1. Go to www.twitter.com and sign in as normal, go to your settings page.
  2. Go to "Widgets" on the left hand side.
  3. Create a new widget for what you need eg "user timeline" or "search" etc.
  4. Feel free to check "exclude replies" if you dont want replies in results.
  5. Now go back to settings page, and then go back to widgets page, you should see the widget you just created. Click edit.
  6. Now look at the URL in your web browser, you will see a long number like this: 441767385733668865
  7. Use this as your id parameter value for the plugin
